# レポート14 — 仮想通貨規制の世界史：日本・米国・EU・アジアの時系列

> 作成日: 2026-05-31
> テーマ: 「規制なし」から始まった仮想通貨が、世界各国でどのように法整備されてきたかを時系列で整理する。
> 対象読者: 規制の流れを体系的に把握したい人（開発者・投資家・学習者）

---

## はじめに — なぜ規制が必要になったのか

ビットコインは 2009年に「政府・銀行不要の通貨」として誕生した。
設計者サトシ・ナカモトのホワイトペーパーには「金融機関への信頼を不要にする」という思想が込められていた。

しかし現実には：

```
規制が必要になった主な理由:
  1. 詐欺・盗難（Mt.Gox, FTX など）
  2. マネーロンダリング・テロ資金供与への悪用
  3. 無許可の証券発行（ICO詐欺）
  4. 税逃れ
  5. 消費者保護の欠如
```

規制は「イノベーションを潰す」ものでも「ブロックチェーンの本質を守る」ものでもなく、
「誰が何に責任を持つか」を社会的に決める作業だ。

## 第1部：日本の規制年表

日本は**世界で最も早く暗号資産を法的に定義した国**の一つだ。
その背景には Mt.Gox という世界最大の事件がある。

---

### 2014年 — Mt.Gox 崩壊：日本発の世界的事件

```
事件の概要:
  取引所:   Mt.Gox（マウントゴックス）
  場所:     東京・渋谷
  規模:     当時ビットコイン取引の約70〜80%を扱う世界最大
  被害:     ビットコイン 85万BTC（当時約480億円）消失
  判明日:   2014年2月（運営停止・破産申請）
  原因:     ハッキング（数年間継続）+ 内部管理の杜撰さ

当時の問題:
  仮想通貨を管理する法律が存在しない
  → 取引所は「無規制のカジノ」状態
  → 顧客の資産保護ルールなし
  → 破産した場合の優先権もなし
```

Mt.Gox 以前の日本では仮想通貨の法的地位すら曖昧だった。この事件が規制立法を急加速させた。

---

### 2016年 — 資金決済法改正：世界初の仮想通貨法整備

```
改正内容（2017年4月施行）:
  ✅ 「仮想通貨」を法的に定義（初めて法律に登場）
  ✅ 仮想通貨交換業者の登録制度（金融庁への登録義務）
  ✅ 利用者資産の分別管理義務
  ✅ 本人確認（KYC）義務
  ✅ 資本金 1,000万円以上の要件

当時の国際的評価:
  主要国でこれだけ明確に法整備したのは日本が世界初
  → ビットコインを「法定通貨に代わる支払手段」として認定
  → 「日本は仮想通貨に最も友好的な国」とみなされた

同時期の課題:
  登録業者（みなし業者）が急増 → 審査が追いつかない
  → 2017〜2018年のバブル期に乱立
```

---

### 2018年 — Coincheck ハッキング事件と規制強化

```
事件の概要（2018年1月）:
  取引所:   Coincheck（コインチェック）
  被害:     NEM（ネム）約580億円相当盗難
             ※Mt.Gox を超える史上最大の取引所ハッキング被害額
  原因:     ホットウォレット（インターネット接続）にNEMを保管
             マルチシグ（多重署名）未使用
             セキュリティ監査不足

金融庁の対応:
  → Coincheck に業務改善命令
  → 全国の仮想通貨交換業者に立入検査
  → みなし業者（申請中の業者）の新規受付を実質停止
  → 登録審査を大幅強化

被害者への対応:
  Coincheck は自社資産（約460億円）で自主補償（全額返還）
  → 法的義務ではなく自主的対応
  → 「補償できるほど会社に資金があった」こと自体が問題視された
```

---

### 2019年 — 金融商品取引法改正：デリバティブ規制

```
改正のポイント（2020年施行）:
  ✅ 仮想通貨デリバティブ取引（先物・CFD）を規制対象に
  ✅ 暗号資産の名称統一：「仮想通貨」→「暗号資産」
     （virtual currency → crypto asset：国際的呼称に統一）
  ✅ ICO（Initial Coin Offering）規制
     → トークンが「有価証券」に該当する場合は金融商品取引法
  ✅ 無登録業者への罰則強化
  ✅ 信用取引の証拠金規制（レバレッジ2倍規制：翌年施行）

レバレッジ規制の詳細:
  改正前: 25倍レバレッジが一般的
  改正後: 個人は2倍まで（2020年5月施行）
  → 海外取引所への流出が加速する一因に
```

---

### 2022年 — ステーブルコイン規制・TerraUSD 崩壊の影響

```
TerraUSD 崩壊（2022年5月）の日本への影響:
  日本人投資家にも多数の被害
  → 「アルゴリズム型ステーブルコインのリスク」が社会問題化
  → 規制立法の議論が加速

FTX 日本法人（FTX Japan）の特殊なケース:
  FTX 本体（バハマ）破綻後、FTX Japan（2022年11月）：
  → 日本の規制により顧客資産は分別管理されていた
  → → 日本ユーザーは他国ユーザーより先に出金可能に
  → 日本の「分別管理義務」の有効性が国際的に証明された
```

---

### 2023年 — 改正資金決済法施行：ステーブルコインの法制化

```
改正内容（2023年6月施行）:
  ✅ ステーブルコインの定義と規制
     → 「電子決済手段」として資金決済法に追加
  ✅ 発行できる主体の限定
     → 銀行・資金移動業者・信託会社のみ
  ✅ 円・外国通貨連動ステーブルコインの取り扱いルール
  ✅ 海外発行ステーブルコイン（USDC等）の特例対応

JPYC への影響:
  現在: 前払式支払手段（改正前のカテゴリ）のまま継続
  将来: 電子決済手段への移行を検討中
       → 電子決済手段になれば「円への返還請求権」が法的保証

FATFトラベルルール完全施行（2023年）:
  暗号資産送金時に送金者・受取人情報の収集・共有義務
  取引所間の情報伝達インフラ（TRUST・Sygna 等）が必要に
```

---

### 2024〜2026年 — 税制・NFT・Web3 の法整備

```
税制の変化:
  従来: 暗号資産の利益は「雑所得」→ 最大55%課税
  2024年議論: 申告分離課税（20%）への変更議論が活発化
              → 2026年時点で実現には至っていないが継続議論

NFT・トークンの分類（2023〜2024年）:
  金融庁・国税庁がNFTの課税・法的分類についてガイドライン整備
  「ユーティリティトークン」「セキュリティトークン」「NFT」の分類基準

Web3 政策:
  自民党 Web3 プロジェクトチーム（NFT ホワイトペーパー 2023）
  → 「日本をWeb3先進国に」という政策方針
  岸田政権の「デジタル田園都市国家構想」にWeb3を組み込み
```

## 第2部：米国の規制年表

米国は「世界最大の暗号資産市場」でありながら、規制の枠組みが複数機関に分散し、今なお整理中だ。

---

### 2013年 — FinCEN ガイダンス：世界初の本格的ルール

```
2013年3月:
  FinCEN（金融犯罪取締ネットワーク）がビットコイン取引所に
  「マネーサービス事業者（MSB）登録」を義務付け
  → AML（マネロン防止）・KYC 義務の適用
  → 最初のまともな規制指針として世界が注目
```

---

### 2015年 — ニューヨーク州 BitLicense

```
2015年8月 NYビットライセンス施行:
  ニューヨーク州内で暗号資産サービスを提供する業者に
  州当局（DFS）のライセンスを要求

問題点:
  取得費用: 数億円規模
  申請負担: 書類・コンプライアンス体制の整備が巨大
  → 多くのスタートアップがNYを「撤退」
  → 「ビットリセンス難民」問題（NY州から他州へ移転）

その後の影響:
  他州はより緩やかな規制で「暗号資産フレンドリー州」に
  → Wyoming州: DAO LLC 法制化（2021）
  → Texas・Florida: 積極誘致
```

---

### 2017年 — SEC の DAO レポート：ICO バブルへの警鐘

```
2017年7月 SECのDAOレポート:
  「TheDAO トークンは有価証券に該当する」と判断
  → Howey テスト（投資契約の判断基準）を適用

ICO バブル（2017年）:
  ビットコイン: 約20万円 → 約220万円（2017年）
  Ethereum: 約1,000円 → 約18万円
  ICO 調達額: 2017年で約6,000億円以上
  → 多くは詐欺・無価値なプロジェクト

SEC の追加措置:
  多数の ICO プロジェクトに有価証券法違反として行政措置
  → 「ICO = 証券発行」という認識が広まる
```

---

### 2021年 — 機関投資家参入とBitcoin ETF議論

```
Bitcoin ETF の議論:
  2013年〜Winklevoss 兄弟が最初に申請
  SEC は2013〜2021年の間に複数回否決
  理由: 市場操作リスク・カストディの不明確さ

機関投資家の参入:
  2020: MicroStrategy が法人資産でBitcoin購入（Michael Saylor）
  2021: Tesla が$1.5B分 Bitcoin 購入
  2021: Coinbase の Nasdaq 上場（直接上場）
  2021: CME Bitcoin 先物ETF（ProShares BITO）承認
        ※現物ではなく先物ベースのETF

El Salvador が Bitcoin を法定通貨に（2021年9月）:
  世界初の国家レベルでの法定通貨採用
  → 国際通貨基金（IMF）が強く反対
  → 世界的な話題に
```

---

### 2022年 — FTX 崩壊と規制強化の転換点

```
FTX 崩壊（2022年11月）:
  取引所: FTX（バハマ本拠）
  CEO:   Sam Bankman-Fried（SBF）
  被害:  顧客資産約100億ドル（1.3兆円）が消失
  原因:  顧客資産を関連会社 Alameda Research の投資・損失補填に流用
         → 実質的な横領・詐欺

SBF の逮捕・有罪:
  2022年12月 逮捕
  2023年10月 詐欺・マネロン等7罪状で有罪評決
  2024年3月  懲役25年の判決

規制への影響:
  → 「取引所規制が不可欠」という議論が爆発的に加速
  → 超党派での規制立法の機運が高まる
  → 同時に「分散化された取引所（DEX）の意義」も再評価
```

---

### 2023〜2024年 — SECの積極訴訟とビットコイン現物ETF

```
SEC の積極姿勢（Gary Gensler 体制）:
  ほとんどの暗号資産は「証券」であるとの立場
  
  2023年6月:
    Binance を証券法違反で提訴
    Coinbase を証券法違反で提訴
    多数のアルトコイン（SOL・ADA・MATIC等）を証券と指定

  Ripple vs SEC:
    2020年12月 SEC が XRP を無登録証券販売として提訴
    2023年7月  裁判所「二次市場でのXRP販売は証券でない」と判決
               → 「プログラムセールスは証券でない」という重要判例

Bitcoin 現物 ETF 承認（2024年1月）:
  BlackRock・Fidelity 等 11社の Bitcoin 現物 ETF を同時承認
  初日の取引量: 約46億ドル
  → 機関投資家が直接 Bitcoin に投資できる道が開く
  → Bitcoin 価格が上昇し 2024年3月に約 $73,000 の史上最高値
```

---

### 2025年 — GENIUS Act とトランプ政権の方向転換

```
SEC の体制変化（2025年）:
  Gary Gensler 辞任
  → 新体制は「暗号資産フレンドリー」な規制姿勢へ
  → Coinbase・Binance への訴訟の取り下げや和解が続く

GENIUS Act（Guiding and Establishing National Innovation for US Stablecoins）:
  ペイメントステーブルコインに特化した法律
  → 発行体に連邦ライセンスまたは州ライセンス
  → 1:1 の法定通貨準備義務
  → FDICの預金保護対象外であることの開示

Bitcoin 戦略的備蓄:
  トランプ大統領が「米国の戦略的 Bitcoin 備蓄」を大統領令で検討
  → 国家がBitcoinを保有する時代の到来（議論継続中）
```

## 第3部：EU の規制年表

EU は世界で最も包括的な暗号資産規制「MiCA」を完成させた先駆者だ。

---

### 2018〜2019年 — 各国の個別対応期

```
EBA（欧州銀行監督機構）の警告（2013〜）:
  「仮想通貨は投機的でリスクが高い」
  → 複数回の警告を出すも統一規制なし

各国の対応:
  ドイツ:   Bitcoin を「私的貨幣（Privates Geld）」として認定（2013）
  フランス:  ICO の法的枠組みを整備（2019年）
  マルタ:   「ブロックチェーン島」として積極誘致
  エストニア: E-Residency × ブロックチェーン行政

```

---

### 2020年 — MiCA 草案の公表

```
2020年9月 欧州委員会が MiCA 草案を公表:
  Markets in Crypto-Assets Regulation
  → EU 統一の暗号資産規制フレームワーク
  → 2年以上の議論・修正を経て最終化
```

---

### 2022〜2024年 — MiCA の成立と施行

```
2023年6月 MiCA 正式採択:
  ART（資産参照トークン）規制: 2024年6月 施行
  EMT（電子マネートークン）規制: 2024年6月 施行
  暗号資産サービス業者（CASP）規制: 2024年12月 施行

MiCA の主な内容:
  ✅ EU 全域で一つのライセンスで営業可能（パスポート制度）
  ✅ ステーブルコイン発行体に準備資産の分別管理・開示義務
  ✅ マーケット操作・内部情報取引の禁止（証券市場と同様）
  ✅ CASP（取引所等）のライセンス制度
  ✅ 大規模ステーブルコイン（重要EMT）に ECBの監督権限

影響:
  USDT: MiCA 対応が遅れ → EU内の主要取引所で上場廃止続出（2024年末）
  USDC: MiCA準拠の方向で対応 → EU市場での優位性を確立
  Binance: EU 各国でのライセンス申請を本格化
```

## 第4部：アジアの規制年表

---

### 中国の規制：最も激しい変遷

```
2013年:   人民銀行が金融機関のBitcoin取扱を禁止（個人は可）
2017年9月: ICO を全面禁止
2017年9月: 暗号資産取引所を全面閉鎖命令（OKCoin・Huobi が海外移転）
2019年:   マイニングを「排除すべき産業」に指定（後に撤回）
2021年5月: 再び大規模なマイニング禁止（内モンゴル→四川→全土）
2021年9月: 暗号資産取引・マイニングの全面禁止
           → 中国のBitcoin マイニングシェア: 70% → 0%（数ヶ月で）
           → その後 Kazakhstan・米国・ロシアにマイナーが移転

理由（推測される背景）:
  デジタル人民元（e-CNY）の普及を優先
  資本逃避の防止
  電力消費の抑制
  共産党の通貨支配権の維持
```

---

### 韓国の規制

```
2017〜2018年:
  「김치 프리미엄（キムチプレミアム）」が社会問題に
  → 韓国のビットコイン価格が世界より30〜50%高い状態
  → 韓国の若者が激しく投機

2018年1月:
  法務大臣が「取引所閉鎖を検討」と発言 → 価格暴落
  → その後「閉鎖しない」と訂正

2021年:
  実名確認義務（銀行口座と取引所アカウントの紐付け）
  → 多くのアルトコイン取引所が廃業

2023年:
  仮想資産利用者保護法 成立
  → 顧客資産の分別管理・不公正取引の禁止
  → 「韓国版 MiCA」として注目
```

---

### シンガポール・香港の「仮想通貨ハブ」競争

```
シンガポール（MAS = 通貨監督庁）:
  2019: Payment Services Act → 暗号資産取引所にライセンス制度
  2022: FTX崩壊後に規制強化
  2023: 個人投資家への暗号資産取引に制限を強化

香港（2023年以降の方向転換）:
  2022まで: 中国の規制に追随・厳格
  2023年6月: 個人投資家への暗号資産取引を解禁（大手取引所にライセンス）
  → 「アジアの暗号資産ハブ」として中国本土とは独立した路線
  → OKX・HashKey 等がライセンス取得
  背景: 中国による間接的な暗号資産市場の取り込み、との指摘もあり
```

## 第5部：FATFトラベルルール — グローバル標準の浸透

単一国の規制ではなく、国際機関による標準が実際の業界に最も大きな影響を与えている。

```
FATF（金融活動作業部会）トラベルルール（2019年勧告）:

  内容:
    1,000ドル（または1,000ユーロ）以上の暗号資産送金で
    送金元・受取人の情報を取引所間で共有する義務

  対象:
    VASP（Virtual Asset Service Provider）＝暗号資産サービス業者

  実装の課題:
    → 取引所間のデータ共有インフラが存在しない
    → TRUST（Coinbase等が主導）、Sygna、VerifyVASP等の
       業界標準プロトコルが乱立

  日本の対応:
    2023年: 国内取引所に完全適用
    → 自国内送金は対応済み
    → 海外取引所との間は継続的な課題

  影響:
    「プライバシー特化コイン（Monero・Zcash等）」の
    取引所取り扱い停止が続く
    → 透明性の高いトークン（BTC・ETH・ERC-20）が規制対応しやすい
```

## 第6部：日本・海外の規制比較まとめ

```
規制の「厳しさ」比較（ユーザー・業者目線）:

  最も厳しい:  中国（全面禁止）
  厳しい:      韓国（実名義務・税制）、米国（SEC 積極訴訟）
  中程度:      日本（分別管理・KYC・レバレッジ規制）
  緩め:        シンガポール（ライセンス制だが成長志向）
  フレンドリー: UAE・El Salvador・マルタ（積極誘致）

日本の特徴:
  ✅ 世界最速の法制化（2017年）
  ✅ 分別管理義務 → FTX崩壊時に日本ユーザーを守った
  ✅ レバレッジ2倍規制（消費者保護 ↔ 海外流出）
  ⚠️ 税制（雑所得55%）が未解決 → 投資家・起業家の海外移住を招く
  ⚠️ 法人の暗号資産保有に時価課税 → Web3企業の参入障壁
```

## 第7部：これから来る規制の焦点（2026〜2030年）

```
焦点 1: DeFi の規制
  現在: 分散型取引所（Uniswap 等）に「管理者」がいないため規制が難しい
  方向性: スマートコントラクトのデプロイ者・フロントエンド運営者を規制対象に

焦点 2: NFT の法的地位
  現在: 国によって「アート」「有価証券」「ゲームアイテム」と解釈が分かれる
  方向性: ユーティリティ・コレクタブル・セキュリティの3分類への整理

焦点 3: AI × 暗号資産
  AI エージェントが自律的に暗号資産を管理・送金するケース
  → 「法的主体性のないAIが資産を操作する」の法的整理が急務

焦点 4: 税制の国際標準化
  OECD CARF（Crypto-Asset Reporting Framework）:
  → 各国税務当局が暗号資産の情報を自動交換する枠組み
  → 2027年ごろ本格実施予定
  → 「海外取引所で脱税」が事実上不可能になる

焦点 5: 日本の申告分離課税移行
  現在（雑所得55%）から申告分離課税（20%）への変更
  → Web3 / 暗号資産産業の国内残留・誘致に直結
  → 2026〜2027年の税制改正で実現する可能性が高い
```
